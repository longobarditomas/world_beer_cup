# World Beer Cup


## API

### Technologies

- PHP.
- Laravel (8.x).

### Getting Started

- Open the api folder in the console.
- composer install
- In the .env file configure your database data. 
- php artisan migrate
- php artisan db:seed
- php artisan serve


## Frontend

### Technologies

- Javascript.
- ReactJS.
- Redux.

### Getting Started

- Open the frontend folder in the console.
- npm install
- npm start
